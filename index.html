Natural Evolution

Software Design final project at Olin with the goal of creating an ecosystem which naturally selects viable genomes.

View on GitHub Download .zip Download .tar.gz

Project Description

Our project Natural Evolution is a simulation of a natural selection and speciation, using genetic algorithms and neural networks. It shows the evolution of organisms that seek to eat food and reproduce “at will”. The simulation is run on Pygame, and the organisms make decisions using neural networks. As the generations progress further, the organisms begin to exhibit interesting behavior.

User Instructions

Our game has the following dependencies: Python 2.7 Pygame 1.9.1 – get it Numpy 1.10.4 – get it To install our simulation game, run the following command in terminal: $ git clone https://github.com/DuncanDHall/NaturalEvolution.git $ python main.py At this point, a PyGame window should appear displaying 10 white "blobs" and one orange "food", which begin interacting. Pressing space stops the program and closes the window.

Implentation
