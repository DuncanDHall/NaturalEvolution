User Instructions

Our game has the following dependencies:
Python 2.7
Pygame 1.9.1 – get it
Numpy 1.10.4 – get it
To install our simulation game, run the following command in terminal:
$ git clone https://github.com/DuncanDHall/NaturalEvolution.git
$ python main.py
At this point, a PyGame window should appear displaying 10 white "blobs" and one orange "food", which begin interacting. Pressing space stops the program and closes the window.

Project Evolution

Prior to learning about neural networks, we created .
The final milestone in our project was implementing natural selection over artificial selection. In the first stages of our project, we would evaluate the fitness of each blob in the generation according to a formula we devised based on food. We then took the two most fit blobs and averaged their features while randomizing it to a certain percentage. This process is called artificial selection, and it was one of our original goals to move past this.
We switched to natural selection by assigning each blob an energy attribute that would decrease as the blob moved or waited, which essentially simulates metabolism in real organisms. If a blob runs out of energy, it dies. However, rather than taking the two most fit blobs at the end of each generation and combining their genes to produce the new generation, blobs would reproduce when they bumped into each other. This mimics natural reproduction and it introduces the need for a whole different style of behavior.
